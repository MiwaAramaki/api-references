# Create Cell

### Overview

This API creates a new Cell

### Required Privileges

Only unit users permitted

### Restrictions

* General Restrictions
    * None

* OData Restrictions
    * Always handles Content-Type in the request header as application/json
    * Only accepts the request body in the JSON format
    * Only application/json is supported for Content-Type in the request header and the JSON format for the response body
    * Response body data is not ensured if atom or xml is specified in the $format query option, although it does not result in an error

<br>

#### Request URL

```
/__ctl/Cell
```

#### Request Method

#### Request Body

##### Format

JSON

|Item Name<br>|Overview<br>|Effective Value<br>|Required<br>|Notes<br>|
|:--|:--|:--|:--|:--|
|Name<br>|Cell name<br>|Number of digits: 1 - 128<br>Character type: Single-byte alphanumeric characters, hyphens ("-"), and underscores ("\_")<br>However, the string cannot start with a hyphens ("-"), and underscores ("\_")<br>|Yes<br>|<br>|

#### Request Sample

```JSON
{"Name":"{CellName}"}
```
